Termometro Sharepoint WebPart

Privacy Statement

The "Termometro" Web Part for Sharepoint, do not collect any user information. It's just a visual web part that show a graphic component.

There is not data communication other than passed in parameters like width, height, text and values used to build the indicator. This component do not connect to any database or any other data collector.

This application do not use cookies and do not save any user data.
