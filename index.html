Horizontal Thermometer Indicator.

This is a Thermometer Indicator (animated) to be included in your web pages. But this time we made it horizontal for best fit in most of the web designs. With this practical 2x1 dimensions is more easy to include as a web parts, like a block in-line between your text.

See it alive (DEMO).

This and ideal part to show advance indicators KPI's with porcentual calculation based on a target goal. And it's animated!! Cool!

Instructions.

For include it in your web pages, you only need to copy-paste the Canvas and Script objects of the html sample page include it in the download. You can change the size of the canvas object, but you need to keep the relation 2x1 between width and height. And we recommend at least 200 x 100 px to see all details. Be sure the termoh.js file is included in the js folder. But if the javascript is in a different location for your site, you only need to make the proper adjustment in the src attribute of your script object, at the html page.
